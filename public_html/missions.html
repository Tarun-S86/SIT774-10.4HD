Missions

Mercury

Mercury missions, such as MESSENGER, have revealed the planet's extreme temperature variations and surface features.

Venus

Venus missions like Venera and Akatsuki have explored its dense atmosphere and extreme surface conditions.

Mars

Mars missions have been critical in studying the planet's geology and the search for past life, including rovers like Curiosity.

Components Used:

This section uses Bootstrap cards to display brief information about space missions for each planet. Each card contains an image of the planet, a short description, and a button that opens a modal with more detailed information about the mission. The modals allow us to present additional content without cluttering the main page.

Implementation Tutorial

Feature: Bootstrap Cards

  1. Include Bootstrap CSS and JS files in the <head> section of your HTML document.
  2. Use the <div class="card"> element to create each card. Include an image using <img> tags and a <div class="card-body"> to hold the content.
  3. Incorporate a button with data-bs-toggle="modal" to link to the corresponding modal for detailed information.

Feature: Bootstrap Modals

  1. Define a modal with the <div class="modal fade"> structure, including modal-dialog and modal-content classes.
  2. Within the modal, use <div class="modal-header"> for the header, <div class="modal-body"> for the content, and <div class="modal-footer"> for buttons to close the modal.
  3. Ensure each modal is linked to its corresponding button on the card using data-bs-target="#modalId".
